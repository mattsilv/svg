# SVG Drop

Paste SVG code from AI tools, preview it instantly, download it as a ready-to-use image.

## Why

Gemini 3.1 Pro (and other AI models) generate amazing SVG files — icons, logos, animated graphics — but give you raw XML with no way to see or use the result. This tool completes the loop.

## How to use

1. Go to [gemini.google.com](https://gemini.google.com/), select **3.1 Pro**
2. Ask it for SVG code (e.g. "give me the SVG for an animated loading spinner")
3. Copy the code, paste it here
4. Pick a preset size (Discord, Slack, GitHub, etc.) and download

## Features

- **Instant preview** with transparency checkerboard
- **One-click presets** — Discord (128px), Slack (512px), Favicon (32px), GitHub (400px), Social (256px), High-res (1024px)
- **Export formats** — PNG, WebP, SVG
- **Options** — transparent background, white background, circle crop
- **Animated SVG** support — previews animations, warns about raster export limitations

## Run it

Open `index.html` in a browser. That's it. Single file, no dependencies, no build step.
